# GitPulse — GitHub 工作仪表盘

在一个集中的工作区中追踪和管理通知、Issue、PR 等，并提供基于 GitHub 搜索的自定义视图，给你更加个性化的体验。

## 特色

### ✨ 自定义视图

利用 GitHub 搜索，来打造你的专属工作区。进入视图管理页面，选择分组创建新视图，即可自由编写搜索语法，并实时查看预览效果。

### 🎯 Todo

通知常常杂乱，无法反映你的真实工作重点。将真正重要的任务加入 Todo 列表，集中处理，不再被干扰。

### 🌐 国际化

内置英文和简体中文，根据浏览器自动检测适合你的语言。

### 🌙 深色模式

适配深色模式，并默认与系统主题保持一致。

## 常见问题

### GitPulse 会存储我的 token 吗？

不会。你的 token 保存在加密的服务端会话 cookie 中——永远不会写入数据库或服务器。
